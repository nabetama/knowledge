# Vimperator

Firefox add-on.


## 設定ファイル作る
:mkvimperatorrc

これで$HOME/.vimperatorrc が作られる

```sh
% cat ~/.vimperatorrc                                                                      [1.9.3-p550]
"3.9 (created: 2015/02/15 15:49:05)

source! /Users/nabetama/.vimperatorrc.local

" vim: set ft=vimperator:
```

## 文字リンクを英字にする

```vim
set hintchars="hjklasdfgyuiopqwertnmzxcvb"
```
